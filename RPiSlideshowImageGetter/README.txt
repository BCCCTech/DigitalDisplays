RPiSlideshowGetter

Image Downloader and Slideshow viewer

These scripts collectively download a set of images(JPGs) from your DropBox
accout and display those images as a slideshow upon boot of your R-Pi.

Modifications to Raspbian:
* install matchbox window manager
* install geeqie (image viewer)

Assumptions:
* user has a ~/slideshow directory into which images will be downloaded
* user has a dropbox account with /Pictures/slideshow directory from
    which images can be downloaded
